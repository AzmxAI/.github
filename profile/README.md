## Quick start

```bash
# macOS — Apple Silicon
brew install --cask azmx-ai

# Linux — AppImage
curl -sSL https://azmx.ai/install | sh

# Windows — installer
# https://github.com/AzmxAI/azmx/releases/latest
```

Open AZMX, finish the 5-step welcome tour, paste a key (or set up free local Ollama), open a folder. **~3 minutes from install to first chat.**

## Security

We take security seriously. The agent has approval gates, a hash-chained tamper-evident audit log, a built-in secret-path screen, and a hard local-only AI mode for total egress lock.

- **Report a vulnerability** → security@azmx.app
- **Security model** → [security.html](https://azmx.ai/security)
- **Compliance** → SBOM (CycloneDX 1.5, signed), SOC 2 pack, DPA on request
